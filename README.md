### Step-by-Step Donut Chart Creation
1. Import the dataset *Superstore_USA* into Tableau.
2. Drag **SUM(Sales)** to Rows *two times*.
3. Apply **Dual Axis** to overlap the charts.
4. Change Mark Type â†’ **Pie** for both axes.
5. Remove color from the second pie and reduce its size to form the inner ring.
6. Drag **Product Category** to Color (first pie only).
7. Add **Sales Labels** to show segment values.
8. Add a **center text label** to show Total Sales.
9. Format chart for better visual appeal (colors, alignment, fonts).
